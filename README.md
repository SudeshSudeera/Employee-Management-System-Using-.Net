# Employee-Management-System-Using-.Net

This application allows an organization or company to manage employees, departments, and employeesâ€™ salaries, tasks, and permissions related to their positions. Users can have an Admin role or a normal user role. The admin role has all authorizations but the user role has limited authority. Admin can assign tasks for different types of users. Each user needs to use their user number and password to log into the system.

This project was created in .Net framework. Windows Forms is used to create user interfaces and MSSQL is for database management. This project was built based on layered architecture.
